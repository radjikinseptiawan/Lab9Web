## Instalasi & Persiapan  
1. Clone repositori:  
   ```bash
   git clone https://github.com/radjikinseptiawan/Lab9Web.git
    ```

2. Masuk ke direktori proyek:

   ```bash
   cd Lab9Web
   ```
3. Instal dependencies (jika menggunakan npm/yarn):

   ```bash
   npm install
   ```

   atau

   ```bash
   yarn install
   ```
4. Jalankan proyek (tergantung setup server lokal):

   * Pastikan server lokal (misalnya XAMPP, MAMP, atau PHP built-in) aktif.
   * Arahkan ke `http://localhost/…` ke folder proyek.
   * Jika menggunakan server PHP built-in:

     ```bash
     php -S localhost:8000
     ```
5. Pastikan database (jika ada) telah dikonfigurasikan sesuai `config` folder.

```

/modules     ← file modul spesifik tiap fitur
```
